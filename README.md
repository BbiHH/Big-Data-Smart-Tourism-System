# Big-Data-Smart-Tourism-System
Big Data Smart Tourism，大数据智慧旅游系统，通过收集互联网上景点评论信息，对景区的服务进行大数据分析。

本项目内容包括 数据爬取、数据处理、前端可视化展示，展示内容为静态页面。

仅供学习，欢迎 star。
